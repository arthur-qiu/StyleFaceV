# StyleFaceV - Official PyTorch Implementation

This repository provides the official PyTorch implementation for the following paper:

**StyleFaceV: Face Video Generation via Decomposing and Recomposing Pretrained StyleGAN3**</br>
[Haonan Qiu](http://haonanqiu.com/), [Hang Zhou](https://hangz-nju-cuhk.github.io/), [Yuming Jiang](https://yumingj.github.io/), [Wayne Wu](https://dblp.org/pid/50/8731.html), and [Ziwei Liu](https://liuziwei7.github.io/)</br>
Arxiv, 2022.

From [MMLab@NTU](https://www.mmlab-ntu.com/index.html) affliated with S-Lab, Nanyang Technological University and SenseTime Research.

## Installation
**Clone this repo:**
```bash
git clone https://github.com/arthur-qiu/StyleFaceV.git
cd StyleFaceV
```
**Dependencies:**

All dependencies for defining the environment are provided in `environment/stylefacev_env.yaml`.
